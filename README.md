# s3-starter

Cyclic S3 file management example:
- upload - using s3 presigned POST
- download - using s3 presigned GET
- list - list bucket contents at prefix
- delete - delete a file by key

Includes:
- Backend:
  - API for s3 ops
- Frontends implemented in:
  - Vanilla JS
  - Vue
  - React
